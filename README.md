# crypto_experiment
## Ex3 RSA大礼包

### 题目描述

已知帧数据的数据格式如下，其中数据都是16进制表示，结构如下：1024bit模数N | 1024bit加密指数e | 1024bit密文c。请给出原文和参数，如不能请给出已恢复部分并说明剩余部分不能恢复的理由？

### 实现的攻击

1. 低指数加密攻击
2. 模不互素攻击
3. 共模攻击
4. 费马分解法
5. Pollard p-1 分解法
6. coppersmith攻击(已知m的高位)，见.sage文件
7. 对随机数发送器的预测..
